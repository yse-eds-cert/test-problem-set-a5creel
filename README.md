# Assignment 1: Introduction
Welcome to ENV 617! This assignment will help you get acquainted with how we'll use GitHub Classroom and Google Colaboratory.

After accepting the assignment, you should now have your own repository named `Assignment-1-Introduction-yourname`. A repository is like a folder that contains a bunch of files. You'll have one repository for each assignment in this course (GitHub will automatically create each repository as you accept each assignment). To start with, each repository will contain a few different files:
* A README with an overview of the assignment (that's this file!)
* A Colab notebook with some starter code

For this assignment, open the assignment notebook (`ENVS_617_Assignment_1_Introduction.ipynb`), and make the specified edits to the file. Once you're done, save, or "commit", your notebook back to this repository.

For detailed instructions on opening the assignment notebook, linking Colab and GitHub, and submitting your assignment, please refer back to the document we shared on Canvas [available here](https://docs.google.com/document/d/1tIySiy1rY7eLUxZgfm-JWS-_mc4F9QcimQSItyn32C0/edit#).
